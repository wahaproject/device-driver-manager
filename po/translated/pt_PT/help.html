Ajuda do Gestor de Driver de Dispositivo

Descrição

O Gestor de Driver de Dispositivo é uma ferramenta para ajudar os utilizadores na instalação de drivers de hardware de terceiros. Atualmente, as drivers da Nvidia, ATI e Broadcom podem ser instaladas e permite instalar o kernel PAE para sistemas de 32 bits multi-core. O Gestor de Driver de Dispositivo só irá listar os dispositivos que são suportados. Ele não irá listar quaisquer outros dispositivos mencionados acima.

Importante

O Gestor de Driver de Dispositivo só pode instalar drivers que estão disponíveis nos repositórios. Verifique se você tem uma ligação com a internet, se você quiser instalar os drivers.

Comandos

Se você preferir instalar os drivers pelo terminal, você pode usar os seguintes comandos:

 

# ddm -i ati

Isto irá instalar os drivers fglrx para a sua placa gráfica ATI. Placas ATI da versão HD 5000 ou superior são suportados pelos drivers fglrx. Versões entre 1000 e 5000 vai usar os drivers Radeon. Use "ddm -p ati" para remover os drivers do seu sistema.

# ddm -i nvidia

Isto irá selecionar os drivers corretos para a sua placa gráfica Nvidia. Ele irá selecionar os drivers Bumblebee no caso de você ter um gráfica híbrida (ambos Nvidia e Intel). Use "ddm -p nvidia" para remover os drivers do seu sistema.

# ddm -i open

Você pode usar este comando se você quiser reverter para os drivers Nouveau abertos. Ela irá remover qualquer drivers proprietários de seu sistema.

# ddm -i broadcom

Este comando irá instalar o driver correto para seu adaptador wireless Broadcom. Use "ddm -p broadcom" para remover os drivers do seu sistema.

# ddm -i pae

Isto irá instalar o kernel PAE em sistemas com múltiplos processadores que executam um sistema operativo de 32-bit. Use "ddm -p pae" para remover o kernel PAE do seu sistema. Observe que você não pode remover o kernel, estando ele atualmente iniciado.

# ddm -i fixbumblebee

Isso irá tentar corrigir a instalação Bumblebee.
