Ayuda sobre el Administrador de Drivers

Descripción

El Administrador de Drivers es una herramienta para ayudar a los usuarios a instalar drivers para hardware de terceros. Actualmente, permite la instalación de drivers de Nvidia, ATI y Broadcom, y el kernel PAE para sistemas multi-núcleo de 32-bits. El Administrador de Drivers listará solo aquellos dispositivos soportados. No listará ningún otro dispositivo aparte de los mencionados anteriormente.

Importante

El Administrador de Drivers solo puede instalar drivers disponibles en los repositorios. Asegúrese de tener conexión a internet si quiere instalar los drivers.

Comandos

Si prefiere instalar los drivers a través de la terminal, puede utilizar los siguientes comandos:

 

# ddm -i ati

Esto instalará los drivers fglrx para su tarjeta gráfica ATI. Las tarjetas gráficas ATI a partir de la versión HD 5000 en adelante son soportadas por los drivers fglrx. Las versiones entre 1000 y 5000 utilizarán los drivers Radeon. Use "ddm -p ati" para quitar los drivers de su sistema.

# ddm -i nvidia

Esto seleccionará los drivers apropiados para su tarjeta gráfica Nvidia. En caso de tener una tarjeta híbrida (Nvidia e Intel al mismo tiempo) se seleccionará el driver Bumblebee. Utilice "ddm -p nvidia" para quitar los drivers de su sistema.

# ddm -i open

Puede utilizar este comando para volver a los drivers Nouveau. Ésto quitará cualquier driver propietario de su sistema.

# ddm -i broadcom

Éste comando instalará los drivers apropiados para su adaptador wireless Broadcom. Utilice "ddm -p broadcom" para quitar los drivers de su sistema.

# ddm -i pae

Ésto instalará el kernel PAE en sistemas multi-procesador que ejecuten un SO de 32-bits. Utilice "ddm -p pae" para quitar el kernel PAE de su sistema. Nótese que no podrá remover el kernel si éste fue iniciado con el sistema.

# ddm -i fixbumblebee

Ésto intentará reparar una instalación de Bumblebee.
