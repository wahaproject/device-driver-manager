Ajuda del Device Driver Manager

Descripció

The Device Driver Manager és una eina per ajudar als usuaris a instal·lar controladors de dispositiu de tercers. Actualment es poden instal·lar controladors Nvidia, ATI i Broadcom; també permet instal·lar el kernel PAE per sistemes de 32 bits amb diversos nuclis. El Device Driver Manager només mostrarà els dispositius compatibles. No mostrarà altres dispositius que els abans esmentats.

Important

El Device Driver Manager només instal·larà controladors disponibles als dipòsits. Assegureu-vos que teniu connexió a Internet si voleu instal·lar els controladors.

Ordres

Si preferiu instal·lar els controladors des del terminal, podeu usar aquestes ordres:

 

# ddm -i ati

Això instal·larà els controladors fglrx per les targes gràfiques ATI. Les targes ATI a partir de la versió HD 5000 són compatibles amb els controladors fglrx. Les versions entre la 1000 i la 5000 poden usar els controladors Radeon. Useu "ddm -p ati" per eliminar els controladors del vostre sistema.

# ddm -i nvidia

Això triarà els controladors correctes per a la vostra tarja gràfica Nvidia. També seleccionarà els controladors Bumblebee en el cas que tingueu una tarja híbrida (amb Nvidia i Intel). Useu "ddm -p nvidia" per eliminar els controladors del vostre sistema.

# ddm -i open

Podeu usar aquesta ordre per a reinstal·lar els controladors lliures Nouveau. Això eliminarà els controladors propietaris del vostre sistema.

# ddm -i broadcom

Aquesta ordre instal·larà el controlador adequat per al vostre adaptador sense fils Broadcom. Useu "ddm -p broadcom" per eliminar aquests controladors del vostre sistema.

# ddm -i pae

Això instal·larà el nucli (kernel) PAE en sistemes multiprocessador amb S. O. de 32 bit. Useu "ddm -p pae" per eliminar el kernel PAE del vostre sistema. Tingueu en compte que no es pot eliminar el nucli que s'està executant en aquell moment.

# ddm -i fixbumblebee

Això intentarà esmenar una instal·lació de Bumblebee.
