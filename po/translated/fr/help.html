Aide du gestionnaire de pilotes de périphériques

Description

Le gestionnaire de pilotes de périphériques est un outil destiné à aider les utilisateurs à installer des pilotes propriétaires. Actuellement les pilotes Nvidia, ATI et Broadcom peuvent être installés, de même que le noyau PAE pour les systèmes 32-bit multi-coeur. Le gestionnaire listera uniquement les périphériques supportés.

Important

Le gestionnaire de pilotes de périphériques ne peut installer que les pilotes présents dans les dépôts. Assurez-vous d'avoir une connection internet fonctionnelle si vous voulez installer ces pilotes.

Commandes

Si vous préferez installer des pilotes via le terminal, vous pouvez utiliser les commandes suivantes:

 

# ddm -i ati

Ceci installera les pilotes fglrx pour votre carte graphique ATI. Les cartes ATI version HD 5000 et supérieures sont supportées par ces pilotes. Les version 1000 à 5000 utiliserons les pilotes Radeon. Utilisez la commande "ddm -p ati" pour supprimer les pilotes de votre système.

# ddm -i nvidia

Ceci sélectionnera le bon pilote pour votre carte graphique Nvidia. Le pilote Bumblebee sera sélectionné si vous utilisez une carte hybride (à la fois Nvidia et intel). Utilisez la commande "ddm -p nvidia" pour supprimer ces pilotes de votre système.

# ddm -i open

Vous pouvez utiliser cette commande si vous souhaitez revenir à l'utilisation des pilotes libres Nouveau. Tous les pilotes propriétaires seront supprimés de votre système.

# ddm -i broadcom

Cette commande installera le bon pilote pour votre carte sans fil braodcom. Utilisez la commande "ddm -p broadcom" pour supprimer ce pilote de votre système.

# ddm -i pae

Ceci installera le noyau PAE pour les systèmes multi-processeur exécutant un système d'exploitation 32-bit. Utilisez la commande "ddm -p pae" pour supprimer ce noyau de votre système. Vous ne pourrez cependant pas supprimer ce noyau si vous l'avez utilisé pour démarrer votre système et qu'il est en cours d'utilisation.

# ddm -i fixbumblebee

Cela va tenter de réparer une installation par Bumblebee.
