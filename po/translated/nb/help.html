Hjelp for driverbehandler

Beskrivelse

Device Driver Manager er et verktøy til å hjelpe brukere å installere tredjeparts maskinvaredrivere. For øyeblikket kan Nvidia, ATI og Broadcom-drivere bli installert. PAE-kjernen kan også installeres på 32-bit multi-kjernesystemer. Device Driver Manager vil kun liste disse støttede driverne. Det vil ikke liste andre enheter enn det som er nevnt ovenfor.

Viktig

Device Driver Manager kan kun installere drivere som er tilgjengelig i arkivet. Pass på at du har internettforbindelse hvis du vil installere driverne.

Kommandoer

Hvis du foretrekker å installere driverne via en terminal, kan du bruke følgende kommandoer:

 

# ddm -i ati

Dette vi installere fglrx driveren for ATI grafikkkortet ditt. ATI kort fra versjon HD 5000 og oppover er støttet av fglrx driverne. Versjoner mellom 1000 og 5000 vil bruke Radeon drivere. Bruk "ddm -p ati" for å fjerne drivere fra systemet ditt.

# ddm -i nvidia

Dette vil velge de riktige driverne for ditt Nvidia-kort. Det vil velge Bumblebee-drivere dersom du har et hybrid-kort (både Nvidia og Intel). Bruk "ddm -p nvidia" for å fjerne driverne fra systemet.

# ddm -i open

Du kan bruke denne kommandoen hvis du ønsker å tilbakestille systemet til de åpne Nouveau-driverne. Det vil fjerne alle proprietære driver fra systemet ditt.

# ddm -i broadcom

Denne kommandoen vil installere den riktige driveren for ditt trådløse Broadcom-adapter. Bruk "ddm -p broadcom" for å fjerne driverne fra systemet ditt.

# ddm -i pae

Dette vil installere PAE-kjernen på multi-prosessor systemer som kjører et 32-bits OS. Bruk "ddm -p pae" for å fjerne PAE-kjernen fra systemet. Merk at du ikke kan fjerne kjernen når du har startet opp med denne.

# ddm -i fixbumblebee

Dette vil prøve å fikse Bumblebee-installasjonen.
