Помощь по менеджеру драйверов

Описание

Менеджер Драйверов Устройств является инструментом, помогающим пользователям устанавливать сторонние драйвера оборудования. На данный момент возможна установка NVIDIA, ATI и Broadcom драйверов, а так же установка pae-ядра для многоядерных 32-разрядных систем. В Менеджере Драйверов будет только список тех устройств, которые поддерживаются Вашим оборудованием.

Важный

Менеджер Драйверов может установить драйвера, только которые доступны в репозиториях. Убедитесь, что у вас есть интернет соединение если вы хотите установить драйверы.

Команды

Если вы предпочитаете установить драйверы с помощью терминала, вы можете использовать следующие команды:

 

# ddm -i ati

Это установит fglrx драйверы для вашей графической карты ATI. ATI карты от версии HD 5000 и выше поддерживаются драйверами fglrx. Версии между 1000 и 5000 будут использовать открытые драйверы Radeon. Используйте "DDM -p ATI" для удаления драйверов из системы.

# ddm -i nvidia

Это позволит выбрать правильные драйверы для видеокарты NVIDIA. Он подберет Bumblebee в случае, если у вас гибридная карта (как NVIDIA и Intel). Используйте "ddm -p nvidia" чтобы удалить драйверы из системы.

# ddm -i open

Вы можете использовать эту команду, если вы хотите вернуться к открытым драйверам. Это удалит любые проприетарные драйверы из вашей системы

# ddm -i broadcom

Эта команда установит нужный драйвер для вашего беспроводного адаптера Broadcom. Используйте "ddm -p broadcom" для удаления драйвера из системы.

# ddm -i pae

Это позволит установить pae-ядро для многопроцессорных систем под управлением 32-разрядной ОС. Использовать "ddm -p pae", чтобы удалить ядро pae из вашей системы. Обратите внимание, что нельзя удалить ядро, когда вы находитесь в сессии запущенной в нем в текущий момент.

# ddm -i fixbumblebee

Это может исправить Bumblebee.
